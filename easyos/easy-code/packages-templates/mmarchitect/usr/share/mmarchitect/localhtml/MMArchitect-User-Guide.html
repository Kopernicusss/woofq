Mind Map Architect

    User Guide

    ! This is still draw with short keys and mouse commands only.

    Idea points

    Idea points could have this meanings:
    • sum - mandays
    • avg - difficulty
    • min - priority

    Mouse and Key commands

    Mouse Right - Context Menu

    ! There is no full mouse support yet, drag & drop are in plan.

    Ideas and branches

    Arrow or Mouse Left - Select active idea

    Enter - Insert new idea next to actual
    Insert - Insert new branch to acual idea to first position
    F2 - Base edit idea title
    2 * F2 - Extend edit idea (note and color)

    + - Expand actual idea branches
    - - Collapse actual idea branches
    Ctrl - Up - Move Up Idea in actual branches list
    Ctrl - Down - Move Down Idea in actual branches list
    Ctrl - Left or Ctrl - Right - Move Idea to parent's branch or from parent's branches next to it
    Mouse 2 * Left - Expand collapsed or collapse expanded branches

    Del - Delete idea with all branches
    Ctrl - c - copy idea with all branches
    Ctrl - x - cut idea with all branches
    Ctrl - v - paste idea with all branches

    Files and tabs

    Ctrl - n - create new file
    Ctrl - s - save file
    Ctrl - Shift - s - save file as
    Ctrl - w - close file

    Ctrl - p - print file

    Alt - Home - welcome tab
