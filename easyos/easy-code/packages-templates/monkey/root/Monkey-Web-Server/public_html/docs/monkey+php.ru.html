Monkey + PHP

Перед установкой PHP, Monkey HTTPD желательно установить зарание.

PHP нужно установить отдельно в систему, даже если он уже был собран и установлен с Apache или другим подобным HTTPD.

Если PHP уже установлен отдельно в систему перейдите к шагу "Конфигурация monkey.conf"

  • Где взять PHP:
      На офицальном сайте PHP ты можешь найти последнюю версию этого прекрасного языка.

      Также можно установить PHP из .rpm для люникс, идущих вместе с вашим дистрибутивом.
  • Установка PHP вместе с Monkey HTTPD:
      В консоли набрать :

        # tar zxfv php-4.x.x.tar.gz
        # cd php-4.x.x
        # ./configure --prefix=/files/my_home/php/
        # make
        # make install

      Эти действия установят PHP в директорию /files/my_home/php/. Если нужна подержка SQL database MySQL,
      добавти опцию --with-mysql к команде ./configure. Также можно добавить другие опции PHP.
      За более подробной информацией обращайтесь на сетевые ресуры PHP, PHP Club (RU) и конечно Google.
  • Конфигурация monkey.conf :
      После правильной установки PHP, мы должны изменить конфигурационный фаил Monkey HTTPD (conf/monkey.conf)

      В этом файле нужно добавит строку:

        AddScript application/x-httpd-php /files/my_home/php/bin/php php

      После этого не забудьте перезапустить Monkey HTTPD, и тогда можно использовать файлы с расширением .php. Они будут обробатоваться с помощью PHP.
      Если хотите что бы сразу открывался фаил index.php в директории, то добавте к строке "index.php":

        Indexfile index.html index.htm index.php
