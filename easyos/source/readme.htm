pup-tools


The folder 'source' contains compilable source code.
These are core utilities used in Puppy.

Those in the 'bacon' folder need to be compiled with the bacon compiler.

The 'create-src-pkg' script creates a source package named 'pup-tools-<date>.tar.gz' that can be compiled in T2 or OE (or anywhere).

The 'build-pup-tools' script compiles the sources and installs the binary files. This will normally be done in T2 or OE, which will then create a binary tarball named 'pup-tools-<date>.tar.bz2'.

In Woof,  file woof-distro/*/*/*/DISTRO_PKGS_SPECS-* can list 'pup-tools' as a package to be included in the build.

It is planned that this will gradually replace the binaries currently in woof-arch/*/target
